## Installation

```sh
pip install -r requirements.txt
```

## Run Tests

```sh
pytest -v
```

Output coverage report.

```sh
pytest -v --cov=./
```

Output htlm coverage report.

```sh
pytest -v --cov=./ --cov-report=html
```
